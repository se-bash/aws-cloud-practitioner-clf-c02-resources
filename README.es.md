# Recursos para el AWS Certified Cloud Practitioner (CLF-C02)

> 🇺🇸 **Prefer English?** Check out the English version in [README.md](README.md).

Una colección organizada de notas de estudio, recursos, exámenes de práctica y resúmenes para preparar el examen **AWS Certified Cloud Practitioner (CLF-C02)**.

---

## 📌 Información General del Examen
- **Código del examen:** CLF-C02
- **Formato:** 65 preguntas (opción múltiple o respuesta múltiple)
- **Duración:** 90 minutos
- **Puntaje mínimo de aprobación:** 700 / 1000

---

## 🎯 Dominios del Examen
1. **Dominio 1: Conceptos de la Nube** (24%)
2. **Dominio 2: Seguridad y Cumplimiento** (30%)
3. **Dominio 3: Tecnología y Servicios en la Nube** (34%)
4. **Dominio 4: Facturación, Precios y Soporte** (12%)
